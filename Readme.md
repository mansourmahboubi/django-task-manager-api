# Task-Manager-API
A simple task-manager using Django 4.1 and django-ninja for async support.
## Start developemnt
```
git clone repo:TODO
pre-commit 
virtualenv .venv -p /usr/bin/python3.8
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
